## 2 分钟概览

### 这个项目是什么

Sweet Trap 可能是**人类共有的行为模式**：个体理性地为降低自身长期福祉的选项持续背书。当前文献分散在多个领域（urban infrastructure / 鸡娃教育 / 996 职场 / 彩礼婚恋 / 甜食健康 / BNPL 消费等），但**从未被统一为一个通用构念并跨域实证**。

本项目目标：
1. 形式化 "Sweet Trap" 为四原语构念（θ, λ, β, ρ）
2. 在 ≥5 个人类生活域做 within-person FE 识别
3. 跨域 meta-analytic 展示这是一个**通用模式**而非域特异性
4. 福利量化跨域总成本 → 政策含义

### 核心创新

比现有行为经济学构念（nudge / default / loss aversion）**范畴更大**：Sweet Trap 不是一个 bias 而是一个**经济均衡现象** — 个体偏好是理性的，但均衡是次优的。
